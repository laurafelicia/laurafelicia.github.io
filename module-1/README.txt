Module 1 - Generative Art

Overview:
	An art installation on a TTGO T1 display running on ESP32 created using the TFT_eSPI library.
The code is written using C++ on Arduino IDE. This code generates new rectangles moving down a 
randomly generated angle, and then changes to a random color when it hits an edge of the screen. 

Set-up
	1. Download Arduino IDE
	2. Go to Tools > Manage Libraries > Install “TFT_eSPI”
	3. If on a Mac, download the Mac port from 
https://github.com/Xinyuan-LilyGO/LilyGo-T-Call-SIM800/issues/139#issuecomment-904390716
